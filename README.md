# 🐍 Basic-Python

Este repositório funciona como um guia prático e de consulta rápida contendo os comandos, sintaxes e estruturas fundamentais da linguagem Python. 

Manter uma base sólida documentada é essencial para agilizar o desenvolvimento no dia a dia, servindo como ponto de partida antes de mergulhar em projetos mais complexos.

## 📚 O que você vai encontrar aqui

Este repositório aborda conceitos essenciais, incluindo:

* **Sintaxe Básica:** Declaração de variáveis, tipos de dados (strings, inteiros, floats, booleanos) e operadores matemáticos/lógicos.
* **Estruturas de Controle:** Condicionais (`if`, `else`, `elif`) e laços de repetição (`for`, `while`).
* **Estruturas de Dados:** Manipulação de listas, tuplas, dicionários e conjuntos (sets).
* **Funções:** Definição de funções (`def`), passagem de parâmetros e escopo.
* **Manipulação Básica:** Exemplos práticos de entrada e saída de dados (`input`, `print`).
